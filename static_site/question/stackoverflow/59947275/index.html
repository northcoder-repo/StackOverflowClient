Two observations:

First:

You should use single-quotes and double-quotes carefully. Take a closer look at the Thymeleaf documentation, and you will see all the examples are like this:

<div th:text="${value.zoneName}">

Single quotes are for text literals:

<div th:text="${'Your zone name is ' + value.zoneName}">

Second:

Looking this example:

<a href='${value.environmentURL}'>${value.environmentName}</a>

This will display the literal value "${value.environmentName}" because that is what you have between the opening <a> and closing </a> tags.

Instead, you have to place your Thymeleaf directive in the tag as an attribute.

Also, your href element needs to be a Thymeleaf href element!

Cleaning it all up, we get this:

<a th:href="${value.environmentURL}" th:text="${value.environmentName}"></a>
