Use disabled instead of readonly. readonly only affects the editable text value of a field - whereas a checkbox has a state not an editable value.

You don't actually need any specific Thymeleaf variables for this. You can just use the following plain HTML in your Thymeleaf template:

<input type="checkbox" id="randomId" disabled checked>

If you do want to use Thymeleaf expressions, you can use this:

<input type="checkbox" id="randomId" th:disabled="${true}" th:checked="${true}">

These are Thymeleaf boolean attributes.

I would just use plain HTML.

To make checkboxes not editable, you can use the disabled attribute instead of readonly. The disabled attribute prevents the user from interacting with the element altogether, while readonly allows the user to see the current value of the checkbox, but they cannot change it.

So, your code should look something like this:

<input type="checkbox" id="randomId" checked="checked" disabled="disabled"/>

This will create a checkbox that is initially checked and cannot be changed by the user.
