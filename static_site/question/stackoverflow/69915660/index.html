You can use the following:

$(document).ready(function() {

  var table = $('#example').DataTable();
  
  table.rows().every(function (rowIdx) {
    $(this.node()).attr('title', rowIdx);
  });

} );

In your question, when you use this, it is referring to a DataTable row (i.e. the JavaScript object). This is because table.rows() is a DataTables API function which returns DataTables objects.

To access the HTML <tr> node contained in each DataTables row object, you need to use the DataTables node() function.
