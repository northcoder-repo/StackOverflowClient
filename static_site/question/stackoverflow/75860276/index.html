I am using DataTables and searched around to find ways to change number of entries shown (default is 10 entries), and how to use DOM to rearrange the positions of elements around the table.

My first issue is, I am not sure how to run datatables API and DOM in single initialisation.

Individually, both codes work:

DOM

$('#datatable').dataTable( {
              "dom": 'tip'
    
          } );

And API,

 var table = $('#datatable').DataTable();  
 table.page.len( 50 ).draw();

The error message:

DataTables warning: table id=datatable - Cannot reinitialise DataTable. For more information about this error, please see http://datatables.net/tn/3

If someone can direct me how to write this code in a single initialisation.

I also noticed, after I make any attempt with DOM, the pagination element moved down and leaves a gap between the element and the table? What is causing this to happen, and how can I avoid this? Shown on screenshot.

You can set the page length to 50 records per page as follows:

$('#datatable').dataTable( {
  dom: 'tip',
  pageLength: 50
} );

Reference: pageLength

Regarding your sample code:

var table = $('#datatable').DataTable();  
table.page.len( 50 ).draw();

That works for me. So, you would need to provide a MRE which recreates the specific error you are getting.

Here is a demo, which sets the page length to 3 rows per page, using your approach:

$(document).ready(function() {

  var table = $('#datatable').DataTable();
  table.page.len(3).draw();

});
