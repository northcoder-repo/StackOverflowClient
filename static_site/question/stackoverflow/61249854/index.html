Assuming your <a> tags are direct children of the relevant <div> tag, you can use the first-child pseudo-selector:

document.select("div.kCrYT > a:first-child[href]")

If that is not guaranteed, you can use the first-of-type pseudo-selector to get the equivalent effect:

document.select("div.kCrYT > a:first-of-type[href]")

EDIT

Some more details for the above solution, in case it helps other people (I know it did not work for the OP, as noted in their comment).

The jSoup version I used was 1.13.1:

<dependency>
    <groupId>org.jsoup</groupId>
    <artifactId>jsoup</artifactId>
    <version>1.13.1</version>
</dependency>

The code:

private void getDivs() {
    String response = testHtml; // the sample html code
    Document document = Jsoup.parse(response);
    if (document != null) {
        // this works:
        Elements links = document.select("div.kCrYT > a:first-child[href]"); 
        // this also works:
        //Elements links = document.select("div.kCrYT > a:first-of-type[href]");
        for (Element link : links) {
            String linkHref = link.attr("href");
            System.out.println("linkHref: " + linkHref);
        }
    }
}
