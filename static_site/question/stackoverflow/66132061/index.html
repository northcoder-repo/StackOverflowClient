The Problem

The ℃ symbol is a single Unicode glyph.

The DataTables PDF exporter uses the pdfMake utility. That uses the Roboto font by default. You can see this by opening the PDF in Adobe Acrobat Reader, and selecting File > Properties > Fonts.

The Roboto font has a limited set of characters:

link to font tester

The character you want to use is not part of the font set.

Quick Solution

Use the degree symbol (°) which is part of Roboto (as shown above): &deg; followed by a separate C.

More Complicated Solution

Use a font which supports that symbol, such as Arial, or Courier New (I think).

See the answer to this question for further details: Changing font in datatables pdfmaker extension.
