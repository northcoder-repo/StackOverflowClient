To run the Thymeleaf "Good Thymes Virtual Grocery" demo project, you can follow these steps:

  1. Download the zip file from here. Click on the green "Code" button and choose "Download ZIP".

(You can choose to clone the repository instead of downloading the zip file.)

  1. Unzip the zip file (e.g. using 7-Zip or a similar tool).

  2. Open a command window (e.g. the CMD prompt if you are using Windows, or a command shell in Linux, etc.) At the command prompt, cd to the unzipped directory thymeleaf-3.1-master and then to the sub-directory examples/core/thymeleaf-examples-gtvg-jakarta.

  3. At the command prompt in the thymeleaf-examples-gtvg-jakarta directory, run the following command, as instructed here:

mvn clean verify cargo:run

If you do not have Maven installed, then download and install it first, and make sure its bin directory is part of your PATH - otherwise the above mvn command will fail.

  1. You will see the project being built and started. At the end, you will see:
[INFO] [talledLocalContainer] Tomcat 10.x started on port [8080]
  1. Open the following URL in a browser: http://localhost:8080

You should see the demo home page:
